# weather-report
A Web application that shows real time weather information based on user location or search city using HTML,CSS, and JavaScript
## Features
- Real-time weather updates
- Next 24-hour weather forecast
- Air Quality Index (AQI) monitoring
- Current location weather detection
- Sunrise and sunset timings
- Clothing recommendations based on temperature
- Travel suggestions based on weather conditions
- Responsive and attractive UI design
## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
## Project Structure
Smart-Weather-Report/
│── index.html
│── style.css
│── script.js
│── README.md
## How It Works
1. User enters city name or uses current location
2. Application fetches real-time weather data from OpenWeather API
3. Displays temperature, humidity, wind speed, AQI, and forecast details
4. Provides smart recommendations for users
